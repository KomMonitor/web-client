Anzahl Punktobjekte pro Gebietskörperschaft

Auswahl einer punktbasierten Georessource, für die eine Punkt-in-Polygon Analyse durchgeführt wird, um die Anzahl der Punkte pro Raumebenen-Feature zu erhalten. Optional können die Punktdaten anhand einer Objekteigenschaft sowie einem Filterwert dieser Objekteigenschaft gefiltert werden (z. B. Objekteigenschaft: Schulform, Filterwert: Grundschule, Operatoren: gleich/ungleich/enthält). Für numerische Werte lassen sich zudem Wertebereiche spezifizieren (z. B. Objekteigenschaft: Anzahl, Filterwert: 50, Operatoren: <, <=, =, >, >=, !=, Wertebereich)

Berechnung: Summe aller Punkte innerhalb jedes Raumebenen-Features. Optional: Anwenden eines Filters anhand einer Objekteigenschaft

Benötigte punktbasierte Georessource

Skript Code - Vorschaufenster (Anzahl Punkte in Polygon)

Der Skript Code ist vordefiniert und kann nicht editiert werden. Dieses Fenster dient lediglich der Vorschau und Nachvollziehbarkeit der Berechnungsschritte.



Überschreiben relevanter Ziel-Indikator Metadaten (Optional)

KomMonitor erlaubt es, die Berechnungs-Methodik des Ziel-Indikators zu überschreiben. Das Vorschaufenster zeigt den (editierbaren) Textbaustein. Um diesen Text als "Methodik" des Ziel-Indikators zu setzen, aktivieren Sie bitte den entsprechenden Schalter und registrieren das Skript.
